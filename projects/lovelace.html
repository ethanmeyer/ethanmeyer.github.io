Ada Lovelace Math Game

Technologies Used: HTML | CSS | JavaScript

October 2020

This project was done for a hackathon hosted by the University of Virginia Girls Hoo Hack developers club. The game was submitted for the educational track and won first place in its category. Our concept was inspired by Ada Lovelace, a mathematician whose work led her to be considered the first computer programmer. My specific contribution to the project was the addition game that is linked here. Players compete with Ada the computer in a race to 21 points. You can find the full project done with others if you hit the go back button on that page.
