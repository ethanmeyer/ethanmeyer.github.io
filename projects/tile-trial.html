Tile Trial

Technologies Used: GameMaker Studio 2 | Networking | SQL

February 2022

This project is a collaboration between 2 other students and myself for our senior project that will be completed on June 1, 2022. Tile Trial is a multiplayer puzzle game connected over the internet. Each "crate" has 4 sides which need to be matched to other crates. You can see to the left a design of what the game logic looks like as well as how it might be scored. We also plan to implement a leaderboard using SQL. We are making this game within the GameMaker Studio 2 application and will be using GameMaker Language to program the logic of the game.
